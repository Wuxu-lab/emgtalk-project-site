# EMGTalk Project Page

This is a static project homepage template for EMGTalk.

## How to use

1. Upload these files to a GitHub repository.
2. Turn on GitHub Pages in repository settings.
3. Replace placeholders in `index.html`:
   - author list
   - paper/code/video/dataset links
   - teaser/system figure
   - result table
   - BibTeX
4. Put images and videos in the `assets/` folder.

Suggested asset names:
- `assets/teaser.png`
- `assets/system_overview.png`
- `assets/demo.mp4`
